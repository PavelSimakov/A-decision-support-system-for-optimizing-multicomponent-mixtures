# A decision support system for optimizing multicomponent mixtures
Программа предназначена для специалистов в области химической технологии, пищевой промышленности, материаловедения и смежных отраслей, чья деятельность связана с разработкой и оптимизацией многокомпонентных смесей. Программа служит для вычисления оптимального состава смеси по нескольким критериям с использованием методов многокритериальной оптимизации (EDAS, TOPSIS, COPRAS, Quadratic Programming). Исходными данными  являются коэффициенты влияния каждого компонента смеси на ее характеристики, веса важности и типы критериев, а также желаемые соотношения между парами характеристик. 
В программе реализованы:
◦	Визуальное представление оптимального распределения долей компонентов в итоговой смеси.
◦	Вывод отчета с расчетными значениями всех характеристик для полученной смеси.
◦	Графическое представление отклонений фактических соотношений характеристик от заданных целевых значений.

This program is designed for specialists in chemical engineering, food processing, materials manufacturing, and related industries whose activities involve the development and optimization of multicomponent mixtures. It is used to calculate the control composition of mixtures based on traditional criteria using multicriteria optimization methods (EDAS, TOPSIS, COPRAS, quadratic programming). The initial data are coefficients obtained for each component mixture based on its characteristics, weight, and type, as well as the desired relationships between the characteristic parameters.
The program implements:
◦ Visual representation of the distribution of component proportions in the final mixture.
◦ Report output with calculated values ​​of all characteristics of the resulting mixture.
◦ Graphical representation of deviations of actual characteristic ratios from specified target results.
